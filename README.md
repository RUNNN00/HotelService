# HotelService
### Projeto de aprendizado
Este projeto consiste em um programa de serviços para hoteis.

Ele possui registro de Hóspedes.
Suas funções são:
- Mostrar quartos disponiveis
- Alugar quarto
- Disponibilizar quarto
